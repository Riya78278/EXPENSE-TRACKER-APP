# EXPENSE-TRACKER-APP
Allows users to track their expenses and make changes like edit, add and delete.
– A dynamic expense entry functionality, allowing users to add, edit, and delete expenses on the go without page reloads, enhancing user convenience and efficiency.
– Technology Used : JavaScript, CSS, HTML.
